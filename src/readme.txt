EWEB Universal SEO Optimizer - Comprehensive, configurable SEO optimization plugin that can be customized for any WordPress site with project-specific settings.

== Description ==

EWEB Universal SEO Optimizer is a flexible, project-configurable SEO plugin designed to implement comprehensive search engine optimization techniques across different websites. Unlike generic SEO plugins, this tool can be customized for specific projects through configuration settings, making it ideal for SEO agencies, developers, and consultants who work across multiple sites.

The plugin follows EWEB plugin development standards and focuses on Core Web Vitals optimization, technical SEO improvements, schema markup implementation, and meta tag enhancement while allowing for project-specific customizations such as primary keywords, social profiles, and business information for local SEO.

== Key Features ==

* EWEB-compliant naming and structure
* Core Web Vitals optimization (defer JS, async CSS, lazy load images)
* Dynamic title and meta description generation
* Schema markup for articles and websites
* Technical SEO elements (viewport, verification tags)
* Project-specific configuration options
* Local SEO support
* Social media tags (Open Graph, Twitter Cards)
* Admin dashboard for SEO analysis
* Reusable across multiple projects with custom configurations

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/eweb-universal-seo-optimizer` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Navigate to the "EWEB SEO" menu to configure project-specific settings.
4. Customize the plugin for your specific project needs using the configuration options.

== Configuration ==

The plugin includes several configuration options accessible through the admin dashboard:

* Primary Keywords: Set project-specific keywords to target
* Local SEO: Enable local business schema for location-based sites
* Google Analytics ID: Integrate with Google Analytics
* Search Console Verification: Add verification meta tags
* Social Profiles: Connect your social media accounts
