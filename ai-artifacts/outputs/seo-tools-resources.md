# Herramientas y Recursos Útiles para Trabajo SEO

## 🛠️ Herramientas Gratuitas de SEO

### 1. **Google Search Console**
- **URL:** https://search.google.com/search-console
- **Uso:** Monitoreo de indexación, errores de rastreo, performance en búsquedas
- **Beneficios:** Datos oficiales de Google, identificación de errores técnicos, mejora de visibilidad

### 2. **Google Analytics 4**
- **URL:** https://analytics.google.com
- **Uso:** Análisis de tráfico, comportamiento del usuario, conversiones
- **Beneficios:** Métricas detalladas de rendimiento, seguimiento de objetivos

### 3. **PageSpeed Insights**
- **URL:** https://pagespeed.web.dev/
- **Uso:** Evaluación de Core Web Vitals y rendimiento web
- **Beneficios:** Recomendaciones específicas para mejorar velocidad

### 4. **GTmetrix**
- **URL:** https://gtmetrix.com (versión gratuita disponible)
- **Uso:** Análisis de velocidad y rendimiento del sitio
- **Beneficios:** Informes detallados de rendimiento, historial de cambios

### 5. **Screaming Frog SEO Spider**
- **URL:** https://www.screamingfrog.co.uk/seo-spider/ (limitada gratuita)
- **Uso:** Auditoría técnica del sitio, detección de enlaces rotos, problemas de indexación
- **Beneficios:** Identificación de problemas técnicos en masa

### 6. **Ubersuggest**
- **URL:** https://neilpatel.com/ubersuggest/ (limitado gratuito)
- **Uso:** Investigación de palabras clave, análisis de competidores
- **Beneficios:** Ideas de contenido, volumen de búsqueda, dificultad de palabras clave

### 7. **Answer The Public**
- **URL:** https://answerthepublic.com/ (limitado gratuito)
- **Uso:** Investigación de preguntas y temas relacionados con palabras clave
- **Beneficios:** Ideas de contenido basadas en consultas reales de usuarios

### 8. **Yoast SEO (versión gratuita)**
- **URL:** Plugin disponible en WordPress
- **Uso:** Optimización on-page, análisis de contenido, sitemap XML
- **Beneficios:** Análisis de contenido, configuración de meta tags, lectura fácil

## 📊 KPIs de SEO Importantes

1. **Tráfico orgánico** - Visitas provenientes de motores de búsqueda
2. **Posiciones en SERPs** - Clasificación de palabras clave objetivo
3. **Tasa de rebote** - Porcentaje de visitas que abandonan rápidamente
4. **Tiempo en página** - Duración promedio de las visitas
5. **Clics en SERP** - Tasa de clics desde resultados de búsqueda
6. **Conversión orgánica** - Acciones deseadas provenientes de tráfico orgánico
7. **Indexación** - Porcentaje de páginas indexadas correctamente
8. **Core Web Vitals** - Métricas de rendimiento de Google (LCP, FID, CLS)

Estas herramientas y recursos proporcionan una base completa para el trabajo SEO profesional, desde auditorías técnicas hasta optimización de contenido y seguimiento de rendimiento.
